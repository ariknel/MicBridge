Command line
Scriptable.
Automatable.

MicBridge ships with a full CLI so you can list devices, start the bridge with exact flags, or wire it into your session-start scripts. No GUI required.

--list-devices enumerate all WASAPI inputs
--device <id> select input device
--sample-rate override sample rate
--buffer-size tune buffer (default: 256)
--debug verbose logging
cmd.exe — MicBridge
C:\> micbridge --list-devices
Found 3 WASAPI input devices:
[0] Blue Yeti X USB Microphone
[1] Focusrite USB Audio
[2] Realtek HD Audio Mic
 
C:\> micbridge --device 0 --sample-rate 48000 --buffer-size 256
Initialising WASAPI capture...
Opening FlexASIO virtual stream...
✓ Bridge running — 18ms latency
# Ctrl+C to stop
